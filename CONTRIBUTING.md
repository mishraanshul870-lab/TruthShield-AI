# Contributing to TruthShield AI

Thank you for your interest in contributing to TruthShield AI!

We welcome bug reports, feature suggestions, documentation improvements, and code contributions.

## Getting Started

1. Fork this repository.
2. Clone your fork.

```bash
git clone https://github.com/<your-username>/TruthShield-AI.git
```

3. Create a new branch.

```bash
git checkout -b feature/my-feature
```

4. Make your changes.

5. Commit your changes.

```bash
git commit -m "Add new feature"
```

6. Push your branch.

```bash
git push origin feature/my-feature
```

7. Open a Pull Request.

## Coding Guidelines

- Write clean and readable code.
- Keep commits focused and descriptive.
- Preserve the existing project architecture.
- Test your changes before submitting.

## Reporting Bugs

When reporting bugs, please include:

- Operating System
- Browser (if applicable)
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots (optional)

Thank you for helping improve TruthShield AI!
